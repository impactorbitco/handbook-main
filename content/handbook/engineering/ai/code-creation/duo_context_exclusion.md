## Overview

Duo Context Exclusion provides intelligent filtering capabilities to prevent sensitive information from being included in AI processing contexts. This ensures security and privacy while maintaining the effectiveness of AI-powered development tools.

## Objectives

- Allow users to identify and exclude sensitive code content (secrets, credentials, PII)
- Provide configurable exclusion rules and patterns
- Maintain security compliance across all AI features
